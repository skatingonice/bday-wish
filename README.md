## 🚀 How to Use

1. Clone this repository

```bash
git clone https://github.com/skatingonice/bday-wish.git
```

2. Open the project folder

```bash
cd bday-wish
```

3. Run `index.html` in your browser

Or simply enable GitHub Pages and make it live.
